MUI Pandeglang Pilih Rayakan Tahun Baru 2025 dengan Doa Bersama Dibandingkan Pesta Kembang Api

Majelis UIama Indonesia (MUI) Kabupaten Pandeglang mengajak masyarakat untuk merayakan Tahun Baru 2025 dengan kegiatan doa bersama kepada Allah SWT. Demikian disampaikan Ketua MUI Kabupeten Pandeglang, KH Zamzami Yusuf ketika merespons kebiasaan menyalakan kembang api dalam merayakan tahun baru. “Ya dengan memanjatkan doa dan muhasabah introspeksi, tingkatkan tenggang rasa. Insyaallah, pada 31 Desember nanti, kami akan mengadakan istighosah di Masjid Agung Ar-Rahman Pandeglang,” katanya, Sabtu (27/12/2024) Ia juga mengimbau masyarakat untuk mengisi malam tahun baru dengan kegiatan yang lebih bermanfaat dibandingkan menyalakan kembang api atau terpengaruh hal-hal negatif.

“Dibanding ikut terhipnotis hal-hal seperti itu, lebih baik tunjukkan bahwa Islam adalah agama yang penuh toleransi. Islam tidak pernah memaksakan orang lain untuk mengikuti kita. Tidak ada unsur paksaan dalam persoalan agama,” jelasnya. Menurut KH Zamzami, pesta kembang api merupakan bagian dari refleksi budaya masyarakat yang tidak perlu disikapi secara berlebihan. Ia menegaskan pentingnya memahami perbedaan tradisi tersebut dengan nilai-nilai keimanan umat Islam. “Hal itu bagian dari refleksi mereka. Kita tidak perlu terlibat lebih jauh, cukup menyadari bahwa mereka berbeda dari sisi keyakinan dan hakikat,” katanya. Lanjutnya, pesta kembang api saat perayaan tahun baru dianggap sebagai tradisi budaya yang terus berkembang seiring dengan kemajuan informasi. “Dulu informasi sangat terbatas, jadi meskipun ada pesta kembang api, hal itu tidak terlihat seperti sekarang,” tutupnya.
